# AIGC 入门教学教案

## 课程信息

- 课程主题：AIGC 入门基础
- 课程时长：60 分钟
- 课件页数：24P
- 适合对象：零基础学员、内容运营、教师、行政宣传、短视频新手
- 课件文件：`index.html`
- 案例素材：`images/01-libtv-case.png`

## 教学目标

1. 理解 AIGC 的基本概念：人负责目标与判断，AI 负责生成与变体。
2. 认识入门工具地图：Gemini、ChatGPT、豆包、Midjourney、Banana、Imag2、可灵、即梦、Seedance 2.0、剪映、Suno、TapNow、LibTV、Codex。
3. 掌握基础工作流：Brief → Script → Image → Video → Edit → Publish → Review。
4. 能基于一张 LibTV 工作台截图，拆解角色、分镜、脚本、场景和视频素材关系。
5. 建立基础风险意识：事实核查、版权肖像、品牌安全和人工验收。

## 现场演示建议

### 案例任务

基于 `东方奇谭｜天阙之战` 工作流截图，演示如何从复杂工作台中拆出一个可执行的短视频项目。

### 提示词模板

```text
你是影视分镜导演。请基于“东方奇谭｜天阙之战”生成 8 个镜头分镜，
风格为东方武侠、云海楼阁、红黑对比、史诗感。
输出表格：镜头编号、画面、动作、景别、镜头运动、字幕、音效、
用于生图/视频的提示词。
```

### Imag2 配图方向

```text
深色工作台背景，东方武侠题材，红黑高对比，云海楼阁，电影分镜卡片，
角色设定板，连线节点关系，统一电子杂志视觉，16:9，无页面标题和页码。
```

## 课堂练习

每组 8 分钟完成一张创作卡：

- 主题：一个产品、活动或知识点。
- 受众：给谁看，为什么要看。
- 脚本：3-5 个镜头。
- 视觉：一条图像提示词。
- 音乐：情绪、速度、是否人声。
- 检查：事实、版权、是否适合发布。

## 讲课金句

- 工具会变，分工不会。
- 提示词不是咒语，是需求说明书。
- 一张复杂工作台，背后是一条完整生产线。
- 生成越快，越需要最后一步人工验收。
